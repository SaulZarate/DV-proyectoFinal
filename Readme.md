# TurApp
Proyecto final de la carrera de Analista de Sistemas


## Instalación 🔧
```
    Descargar/Clonar el repositorio
```

Reemplazar las variables de configuración
```
    /config/constants.php
    /assets/js/main.js => DOMAIN_NAME
```

* Javascript vanilla
* Bootstrap 5
